# Jishnu Suresh - Personal Portfolio

A clean, minimalist, developer-focused personal portfolio website built with a strict monochrome aesthetic. 

## 🚀 Live Demo

[Link to your live website - e.g., https://jishnu-jsk.github.io/portfolio]

## 🛠️ Built With

*   **HTML5:** Semantic structure and content organization.
*   **CSS3:** Custom styling, CSS Variables, Flexbox/Grid layouts, and CSS animations (no external frameworks like Tailwind or Bootstrap).
*   **JavaScript (Vanilla):** Intersection Observer API for scroll animations, dynamic UI interactions, and responsive mobile navigation.

## ✨ Key Features

*   **Monochrome Aesthetic:** A premium, hacker-minimalist black, white, and gray color palette.
*   **Custom UI Mockups:** Simulated IDE environments, application interfaces, and pipeline diagrams crafted entirely with HTML/CSS.
*   **Performance Focused:** Lightweight and fast, using vanilla technologies without heavy dependencies.
*   **Responsive Design:** Fully fluid layout that adapts seamlessly from mobile to desktop screens.
*   **Scroll Animations:** Smooth reveal animations as elements enter the viewport.

## 📂 Project Structure

*   `index.html`: The main structured document containing the portfolio content.
*   `style.css`: All the styling, variables, layout systems, and animations.
*   `script.js`: Logic for scroll animations (Intersection Observer) and mobile menu toggling.

## 💻 Running Locally

To view the project locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/Jishnu-JSK/your-repo-name.git
   ```
2. Navigate to the project directory:
   ```bash
   cd your-repo-name
   ```
3. Open `index.html` in your preferred web browser, or use a local development server like VS Code's Live Server extension, or Python's built-in HTTP server:
   ```bash
   python3 -m http.server 8000
   ```
   Then visit `http://localhost:8000` in your browser.
